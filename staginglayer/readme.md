# Staging Layer

## Overview

The Staging Layer stores validated and standardized datasets generated from transformation processes.

## Purpose

- Intermediate storage before warehouse loading.
- Ensure data quality.
- Support analytical processing.

## Files

### customers_stage.csv
Validated customer data.

### products_stage.csv
Validated product data.

### orders_stage.csv
Validated transaction data.
